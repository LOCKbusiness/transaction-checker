
Install Node Software:
----------------------
https://nodejs.org


Install Node Modules:
---------------------
npm install --save bitcoinjs-message
npm install --save @defichain/jellyfish-network
npm install --save @types/node


Install Typescript:
-------------------
sudo npm install -g typescript


From .ts to .js:
----------------
npx tsc app.ts


Local start .js:
----------------
node app.js


Make an executable:
-------------------
sudo npm install -g pkg

pkg app.js
